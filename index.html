Bienvenue sur le site du Memory de l'ENI.
Venez jouer avec nous et faites le meilleur score possible!

Règles du jeu:

    Retourner deux cartes:
  • Si elles sont identiques: le jeu laisse les cartes visibles.
  • Si elles sont différentes : les cartes sont retournées faces cachées là où elles étaient.
  • La partie est terminée lorsque toutes les paires sont assemblées
Si vous voulez vous souvenir de votre score : inscrivez vous via le formulaire dédié.
Le jeu garde les 5 meilleurs scores en mémoire.
Si vous voulez relancer une partie, cliquez sur la barre d'espace.
